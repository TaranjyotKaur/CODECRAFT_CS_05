# 📡 Packet Sniffer Tool (Educational Use)

A Python-based network packet sniffer that captures and analyzes live network traffic.

## 📌 Project Overview

This tool captures packets and displays:

- Source IP address
- Destination IP address
- Protocol (TCP/UDP/ICMP)
- Payload data

Built using Scapy for reliable packet capture on Windows.



## 🛠 Technologies Used

- Python 3
- Scapy
- Npcap (Windows)
